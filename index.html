Trusted Tools.

Which applications out there can you trust. Unfortunately, you commonly have trust any application you are installing that it won't steal your data, tell on your whereabouts or simply do things you didn't intend.

Trusted OpenSource applications

  • Syncthing http://syncthing.net ** Awesome direct file syncing between computers.
  • Seafile http://seafile.com ** Run your own data server to store files.
  • OwnCloud http://owncloud.org ** Run your own cloud server to store anything from files to address books.
  • Baikal http://baikal-server.com ** Sync your own personal address book and calendars.
  • GPGtools https://gpgtools.org ** Encrypt anything.
  • TextSecure / Signal iOS app https://github.com/WhisperSystems ** Secure and end-to-end encrypted text messaging on Android & iOS
  • LaTeX http://www.latex-project.org

Recommended applications

These are not trusted or simply not fully available or transparent to the user, but recommended as good alternatives.

  • Heartbeat https://ind.ie

Common applications to STAY AWAY from

We've crafted a small limited list of common applications that many people use, but that you probably want to stay away from. These untrusted tools are common proprietary software and probably don't want to do you any harm. However, it is difficult to verify their internals. STAY AWAY from:

  • Dropbox
  • Box
  • Skype
  • Bittorrent Sync
  • TeamViewer
  • WhatsApp
  • Microsoft Office
